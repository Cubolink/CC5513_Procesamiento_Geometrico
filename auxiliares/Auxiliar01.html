Auxiliar 1¶

El objetivo de este auxiliar introductorio, es familiarizarse con polyscope y poder resolver problemas en el espacio.

Como vieron anteriormente, hay varias formas de representar los datos. Por ejemplo, voxeles, que tienden a ocupar mucho espacio dada su naturaleza. Los voxeles son utilizados en algunas investigaciones con Deep Learning, pero por su tamaño y la cantidad de datos y entrenamiento muchas veces se utilizan con resoluciones muy pequeñas.

Ej: (https://arxiv.org/pdf/1604.00449v1.pdf)

Otra representación común son las mallas geométricas. Las GPU se llevan bien con ellas, ya que trabajan generalmente con triángulos, así que son fáciles de renderizar, detectar colisiones, intersecciones, deformaciones, etc.

Y otra muy común son las nubes de puntos, directamente puntos en el espacio, formando parte y representando algún objeto.

Ej de PointNet (https://arxiv.org/pdf/1612.00593.pdf)

Curvas¶

Polyscope nos permite también dibujar curvas, que viene a ser parecido a dibujar líneas en OpenGL. Necesitamos los puntos en el espacio y cómo se conectan.

Como ejercicio de este auxiliar, dibujaremos un círculo en nuestro espacio 3D. Pero necesitaremos repasar un poco de geometría y álgebra lineal.

Obtener puntos de un círculo¶

Los círculos son objetos que viven en un plano, pero en un espacio 3D este puede estar inclinado, así que nuestro círculo estará definido por 3 valores:

  • Centro
  • Radio
  • Normal al plano del círculo

El centro y radio del círculo nos permitirán entenderlo dentro del plano, y es la normal la que nos permite fácilmente describir el plano perpendicular, pues con ella podemos obtener los dos vectores del plano.

Claro que podríamos directamente definir el círculo entregándole estos dos vectores, pero es más complicado de utilizar a la hora de querer uno manualmente definir círculos.

Obtención de los vectores perpendiculares a la normal¶

Para obtener el primer vector del plano, llamémosle $v_1$, necesitamos que sea perpendicular a la normal. Pero recordemos que el producto punto entre dos vectores perpendiculares es siempre 0, por lo que podríamos generar uno en base a esta regla.

Si $n=(a, b, c)$, entonces basta que $v1=(-b, a, 0)$, pues esto hará que $n\cdot v = 0$. Aunque hay una excepción, si la normal fuera $(0, 0, 1)$ esto no funcionaría, sin embargo en este caso podemos libremente elegir el vector $(1, 0, 0)$, que sabemos que es perpendicular.

Ahora que tenemos $v1$, como sabemos que el producto cruz entre dos vectores perpendiculares produce uno perpendicular a ambos, basta $v_2 = v_1\times n$

Obtención de los puntos¶

Normalmente en 2D, sabemos que podemos obtener puntos de un círculo en base a un ángulo, utilizando $(R\cos\theta, R\sin\theta) = R\cos\theta\hat{i}+ R\sin\theta\hat{j}$. En este caso, como nuestras bases son $v1$ y $v2$, utilizamos simplemente $$ \vec{C} + R\cos\theta\vec{v1}+ R\sin\theta\vec{v2} $$

Notemos que C es para mover el círculo a su centro, el punto C.

# Curve network examplee

import numpy as np
import polyscope as ps

class Circle:
    def __init__(self, center, radius, normal):
        self.c = np.array(center)
        self.r = radius
        self.n = np.array(normal)
        self.n = self.n/np.linalg.norm(self.n)  # normalized here so we don't have to worry
        

def generate_circle_node_edges(circle: Circle, n_nodes=10):
    c, r, n = circle.c, circle.r, circle.n
    
    # let v1, v2, n an orthonormal system
    
    # get v1 so <v1, n> == 0
    v1 = np.array([n[1], -n[0], 0])
    if np.array_equal(v1, np.zeros(3)):
        v1 = np.array([1, 0, 0])
    
    # v2 = n x v1
    v2 = np.cross(n, v1)
    
    # make them orthonormal
    v1 = v1/np.linalg.norm(v1)
    v2 = v2/np.linalg.norm(v2)
    
    # sample
    nodes = []
    edges = []
    for i in range(0, n_nodes):
        theta = 2 * np.pi * (i / n_nodes)
        nodes.append(c + r * (v1 * np.cos(theta) + v2 * np.sin(theta)))
        edges.append([i, (i+1) % n_nodes])
    
    return np.array(nodes), np.array(edges)

our_circle = Circle([0, 0, 0], 10, [0, 1, 1])
circle_nodes, circle_edges = generate_circle_node_edges(our_circle, 20)


ps.init()
ps_cloud = ps.register_point_cloud = ps.register_point_cloud("Supporting Circle Centers", np.array([our_circle.c]))
ps.register_curve_network("Circle", circle_nodes, circle_edges, radius=0.001)
ps_cloud.add_vector_quantity("Normal", np.array([our_circle.n]))

ps.show()

Vemos que adicionalmente a registrar la curva con los nodos y las aristas, registramos el centro del círculo y le asociamos su normal en una cantidad vectorial, para poder visualizar completamente nuestro círculo.
